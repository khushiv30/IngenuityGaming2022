Steps to configure your TypeScript configuration to receive .ts file from src Folder and output .js file in dist Folder.

1. Create a project folder

2. Run command npm install -g typescript in terminal to install typescript

3. To check the version of typescript run tsc -v command on terminal

4. Make a folder with src name

5. Inside src folder create a file with .ts extension (say a.ts)

6. Write some typescript code in it

7. Write tsc --init command in terminal which will create tsconfig.json file

8. open tsconfig.json and change "rootDir": "./", to "rootDir": "./src", "outDir": "./", to "outDir": "./dist"

9. Finally run tsc command in the terminal and dist folder will be created along with a.js file
